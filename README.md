# SHA Payments Dashboard

A stakeholder-focused analytics dashboard built with NiceGUI, Polars, and ECharts to help county health teams explore and analyze SHA PHC payments made to health facilities.

The project was created after scraping payment records from the public SHA portal and transforming them into an interactive dashboard that helps stakeholders answer questions such as:

* Which counties received the highest PHC payments?
* Which sub-counties are receiving the largest share of funds?
* Which facilities are receiving the highest payments?
* How are payments distributed across facility types, ownership categories, and KEPH levels?
* Are there payment concentration risks or outlier facilities?


## Tech Stack

| Tool        | Purpose                         |
| ----------- | ------------------------------- |
| **NiceGUI** | Frontend dashboard framework    |
| **Polars**  | Fast analytical data processing |
| **ECharts** | Interactive charts              |
| **Loguru**  | Application logging             |
| **Python**  | Backend application logic       |
